pygbm documentation
===================

   *Author*: Yuhao Huo

The **PyGBM Package** is a Python package designed to simulate Geometric Brownian Motion using different numerical methods.


Features
--------

- **`GBM_simulator` class**: Core attributes and methods

Installation
------------

Ensure you have Python 3.9 or higher. Install the package and its dependencies with:

.. code-block:: bash

    pip install -e .
